I made a programme that will allow you to create a four-option interactive quiz with a question limit of 100 questions.

Here's how to use it:

There's a button that will display on opening the website (its URL: https://quizforge-t409.onrender.com).
You can create questions and when you're done, click the "Save File" button. This will generate a JSON file that you can use later for the following step.
You can as well load pre-saved questions by clicking the "Load File" button.
On completing either of the above, click the "Storage Key" button.
A pop-up window will display and in the "Public Quiz Page URL" space, simply copy and paste the URL I put above. 
Click Save Settings and Continue when you're done.

You can then click the "Get Share Link" button to generate a link for the test.

That's about it. I'll make expansions later on but this is all for now.
